# Hotel
UI-Kit is published at https://nelliza.github.io/hotel/main.html  
Hotel website is published at https://nelliza.github.io/hotel/

## Clone repository
```
$ git clone https://github.com/Nelliza/hotel.git
```

## Installation
```
$ cd hotel
$ npm i
$ npm run build
```

## Development
```
$ npm run dev
```

## Publication
```
$ npm run deploy
```
